LightnessPair

@objc
public class LightnessPair : NSObject

Represents a pair of lightness variants for a specific color.

Lightness means the L component in the HSL color representation.

Lightness variants are used in pairs, making it easy to handle dark mode for user interfaces.
Requestting a specific lightness variant for a color will automatically return the corresponding value in the pair if dark mode is on.
